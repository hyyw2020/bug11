## 📋 项目简介

**bug11** 是一款功能强大的 Windows 系统映像构建工具，支持对 Windows ISO 进行深度定制和精简。

### ✨ 核心功能

- 🖥️ **ISO 挂载和提取** - 自动挂载 ISO 文件并复制内容
- 🔧 **系统精简** - 删除遥测、蓝牙、搜索等不必要的组件
- 🗑️ **应用清理** - 移除预装应用和更新程序
- 🌍 **语言优化** - 保留指定语言包，删除多余语言
- 📦 **多格式输出** - 支持 ISO、WIM、ESD 三种输出格式
- 🔒 **硬件限制绕过** - 注入 Windows 11 硬件检查绕过配置
- ⚡ **GUI 界面** - 友好的图形化操作界面
- 🛡️ **安全保障** - 完整的错误处理和异常恢复机制

## 🚀 快速开始

### 前置条件

- Windows 10 / Windows 11 (任意版本)
- PowerShell 5.1+ (系统内置)
- **管理员身份运行**
- 系统盘至少 40GB 可用空间

### 一键执行

在 **PowerShell 5.1 (管理员)** 中执行主程序。

### 使用流程

1. **① 选择输入 ISO** - 点击按钮选择 Windows ISO 文件
2. **② 选择输出路径与格式** - 选择输出位置和文件格式 (.iso/.wim/.esd)
3. **③ 勾选精简选项** - 选择要删除的组件
4. **④ 开始构建** - 点击开始按钮，等待完成
